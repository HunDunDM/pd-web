# pd-web

PD Web by PingCAP FE.

## The Rules

**Please ensure that each commit is followed the setting of:**

- `.editorconfig`
- `.prettierrc`

## How to develop

```sh
git clone https://github.com/pingcap-fe/pd-web.git && cd pd-web

# We recommend using yarn
yarn
yarn start
```

## Run test

Make sure the necessary unit tests.

```sh
yarn test
```
